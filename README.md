# 멀티 공포 게임 제작 프로젝트

# 1. 프로젝트 개요
- 프로젝트 이름 : 멀티 공포 게임 제작 프로젝트
- 의도 : 리썰컴퍼니류의 멀티 공포 게임 제작
- 개발 인원 : 2인
- 개발 도구 : Unity 2021.3.42f1 
- 개발 언어 : C#
- 기능 구현 :
    - 네트워크
    - 환경 설정
    - 캐릭터
    - 카메라
    - 아이템
    - 인벤토리
- 총 개발 기간 : 25.02.20 ~ 25.03.28
 
# 2. 게임 규칙 
- 몬스터가 존재하는 맵을 탐험
- 맵에 존재하는 아이템을 파밍
- 파밍한 아이템을 판매하여 재화를 축적하고 생존 기간을 늘리는 게임
